// Mode opératoire

1. Position récepteur : cliquez sur la carte ou saisissez les coordonnées.

2. Azimut : entrez le relèvement magnétique/géographique en degrés (0–360°).

3. Incertitude : la valeur est utilisée dans le calcul, sans affichage du secteur.

4. Longueur du trait : longueur du tracé sur la carte.

5. QRK : force du signal, modifie l'épaisseur du trait de relevé.
